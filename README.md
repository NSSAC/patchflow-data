# PatchFlow 
## Short description
PatchFlow is a set of precalculated networks for simulating the metapopulation SEIR model at a global level

## Versions
|         |             |
| ------- | ----------- |
| 1.0  |  Initial release of the GPW-GADM based admin 1 and admin 2 level regions |
